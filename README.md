# 🛒 Marketplace Digital

Um marketplace completo com funcionalidades de vendedor e consumidor, construído com React + TypeScript (frontend) e Node.js + Express + MongoDB (backend).

## 🚀 Início Rápido

### 1. Verificar Node.js
```powershell
node --version
```

### 2. Instalar Dependências
```powershell
# Backend
cd backend
copy env_final.txt .env
npm install

# Frontend
cd ../frontend
npm install
```

### 3. Executar
```powershell
# Terminal 1 - Backend
cd backend
npm run dev

# Terminal 2 - Frontend
cd frontend
npm start
```

### 4. Acessar
- **Frontend:** http://localhost:5173
- **Backend:** http://localhost:5000
